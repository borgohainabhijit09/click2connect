# Template Images

The template preview images should be placed in this directory.

You can:
1. Create simple preview images using any design tool (Canva, Figma, etc.)
2. Use placeholder images temporarily
3. Generate them programmatically

## Image Specifications

- **Dimensions**: 600x400 pixels (3:2 aspect ratio)
- **Format**: PNG or JPG
- **File names**:
  - modern-blue.png
  - elegant-purple.png
  - professional-green.png
  - creative-orange.png
  - minimal-black.png
  - vibrant-pink.png

## Quick Solution

For now, the component will show color-based previews even without images.
You can add actual template preview images later.

## Creating Template Images

Use Canva or Figma to create simple business card previews showing:
- A circular profile placeholder
- Name section with the template's primary color
- Contact details section with secondary color
- Clean, minimal design

Export as PNG and place in this directory.
